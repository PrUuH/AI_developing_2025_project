# AI M&A Platform — Умная продажа бизнеса

**Автоматическая оценка, поиск покупателей и аукцион для владельцев малого и среднего бизнеса**  
_(до $100 млн выручки)_

> **Проблема**: Традиционные M&A-банки не работают с бизнесами до $100 млн → владельцы теряют миллионы из-за узкого круга покупателей и отсутствия конкуренции.  
> **Решение**: Наша платформа автоматически находит релевантных покупателей, оценивает бизнес и запускает аукцион — всё через ИИ.

---

## Возможности

- **Оценка стоимости** на основе мультипликаторов (аналоги сделок)
- **Поиск покупателей**:
  - Стратегические (конкуренты)
  - Финансовые (фонды)
  - Предприниматели
- **Персонализированные email-предложения**
- **Граф скрытых связей** («Компания X купила в Берлине → может интересоваться вами»)
- **Симуляция аукциона** → повышение цены на **25–40%** при конкуренции
- **Анонимный teaser** → никаких конфиденциальных данных до NDA

---

##  Технологии

- **Python 3.10+**
- **Streamlit** — веб-интерфейс
- **Scikit-learn** — ML-ранжирование
- **NetworkX** — граф связей
- **Hugging Face (ruGPT)** — генерация teaser'ов
- **SQLite** — синтетические данные
- **Pandas, NumPy** — анализ данных
